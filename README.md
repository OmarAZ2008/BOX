BOX

A simple 10-level grid-based game made with Python and Pygame

How to play:
- Use WASD or Arrow Keys to move
- You play as a blue box navigating grid-based levels
- Reach the green goal tile to complete the level
- Press R to restart the level
- Press SPACE to select
- Press ESC to return to previous page

Features:
- 10 handcrafted levels
- Toggleable sound effects
- Save system for unlocked levels and SFX option

How to run:
- Install Python
- Install Pygame
- Run main.py
- You can build using PyInstaller
